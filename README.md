# OTP Notifications

A small bridge for forwarding OTP codes from iOS Shortcuts to a Linux desktop notification.

The iOS Shortcut extracts one-time passcodes from incoming SMS content and sends them to a local FastAPI service.
The desktop service shows the code in a notification and includes a copy action for quickly placing the OTP on the clipboard.
